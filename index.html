Mybookdb

Manage your book library at home for hardcover books

Download this project as a .zip file Download this project as a tar.gz file

Welcome to mybookdb site.

This program stores your informations about books and hold it in a database. You can also adding books by scan the barcode with your Android phone through GCStarScanner app.

This program is under developemnt and in alpha state, some basic functions didn't work.
So check out later on or set this project to your watchlist in github.

screenshots

Clone 

$ git clone https://github.com/vatriani/mybookdb.git

Compile 


$ cd mybookdb
$ mkdir bin
$ cd bin

$ cmake ../
$ make

Run 

$ src/mybookdb
